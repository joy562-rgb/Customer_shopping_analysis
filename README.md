Customer Shopping Behaviour Analysis Dashboard
Project Overview

This project focuses on analyzing customer shopping behaviour using Python, PostgreSQL, and Power BI. The objective of the project is to understand customer purchasing patterns, spending behaviour, subscription trends, product performance, and revenue generation through data analysis and visualization.

The project includes:

Data cleaning and preprocessing using Python
Data storage and querying using PostgreSQL
Interactive dashboard creation using Power BI
Business insight generation using SQL analysis
